#Vorlon.js
>remote debug js的工具


##Install
```sh
$ sudo npm install -g vorlon
```

##Usage
###1. 啟動vorlon server
```sh
$ vorlon
```

###2. 用瀏覽器打開` http://localhost:1337/dashboard/SESSIONID`

###3. 在你的網頁裡面加入
```html
<script src="http://localhost:1337/vorlon.js/SESSIONID"></script>
```
