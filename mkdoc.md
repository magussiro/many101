>[Mkdocs](http://www.mkdocs.org/)是一個使用markdown寫文件放到網站的工具

##How to Start

####[install pip](http://pip.readthedocs.org/en/latest/installing.html#id7)
```sh
$ curl https://bootstrap.pypa.io/get-pip.py > get-pip.py
$ sudo python get-pip.py
```

####install mkdocs
```sh
$ pip install mkdocs
```

####建立你的第一個doc資料夾
```sh
$ cd /path/to/project
$ mkdocs new docs
```

####建立文件網頁
```sh
$ mkdocs build
```
